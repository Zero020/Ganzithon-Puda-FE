## 프로젝트 소개
푸다는 매일 발생하는 베이커리·카페·식당 등의 남는 음식을
지역 복지시설과 연결하여 낭비는 줄이고, 결식 문제를 완화하는 사회적 가치 기반 서비스입니다.

- 남는 음식 → 기부 가능한 자원으로 전환

- 공공데이터(OpenAPI) 기반 기관 인증

- 영수증 OCR(Upstage API)로 기부의 투명성 & 안전성 확보

- 가게 ↔ 복지시설 실시간 매칭·예약 시스템 제공

**본 리포지토리는 푸 서비스의 프론트엔드 리포지토리입니다.
#### 백엔드 리포지토리: https://github.com/Ganzithon-team6/team6-BE


<br><br>

## 문제 상황 & 해결 방식

(1) 남는 음식 연결 부재

- 가게 재고와 시설 수요가 단절되어 동시에 “낭비 + 부족” 발생

(2) 출처 불명·위생 문제

- 개인 중심 공유 플랫폼은 위생성과 출처 검증이 어려움

(3) 기부 흐름 불투명

- 기록 미흡으로 기부 증빙과 수령 관리 불가

#### ✅ 해결책

(1) 실시간 음식 매칭 시스템

- 가게가 등록한 잉여 음식 → 즉시 복지시설과 매칭

(2) 공공데이터 기반 기관 인증

- 사회복지시설정보 OpenAPI로 시설 신뢰도 확보

(3) 기부·수령 자동 기록 시스템

- 예약·픽업·리뷰까지 전 과정을 자동 로그로 기록
